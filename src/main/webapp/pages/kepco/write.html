공지사항

~

체크시 공지사항으로 게시물 최상단에 출력됩니다.(정렬기준: 최근게시물)
게시기간 미 입력시 상시로 게시됩니다.

수정,삭제시 확인 목적으로 사용됩니다.
공란으로 입력시 현재일자로 저장됩니다.
내용*
  • 에디터로 이미지 첨부시 이미지 설명을 적어주시기 바랍니다.
  • jpg나 이미지 파일인 경우 파일설명을 입력해주세요
  • pdf나 hwp 파일이 이미지 기반일 경우 텍스트로 변환후 첨부해주세요!
최대 5개 파일 업로드 가능. 전체 업로드 용량은 50MB 이하
파일설명은 웹표준 및 웹접근성에 의해 입력하는 것을 권장합니다.
